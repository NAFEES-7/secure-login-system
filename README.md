# 🔐 Secure Login System (Python)

A simple command-line based login and registration system built using Python. This project demonstrates file handling, authentication logic, and basic cybersecurity concepts.

## 🚀 Features
- User Registration
- User Login Authentication
- 3 Attempt Login Limit
- File-based credential storage

## 🛠️ Technologies Used
- Python
- File Handling

## 📂 Project Structure
- main.py → Main application
- users.txt → Stores user credentials
- README.md → Project documentation

## ▶️ How to Run
1. Clone the repository
2. Navigate to project folder
3. Run:
   ```bash
   python main.py
